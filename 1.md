
# Date and Time Commands

These commands help us check and manage system date and time.

---

## `date`

Shows current date and time.

### Example

```bash id="c9h3r2"
date
```

### Output

```bash id="9g1z6v"
Thu May 28 10:30:25 IST 2026
```

---

## `cal`

Shows calendar.

### Example

```bash id="g5q9mt"
cal
```

---

## `timedatectl`

Shows system time settings.

### Example

```bash id="r0u6ne"
timedatectl
```

# Network Commands

These commands help check network connectivity.

---

## `ip addr`

Shows IP address.

### Example

```bash id="4q9u3h"
ip addr
```

---

## `ping`

Checks internet or server connection.

### Example

```bash id="1x8vbe"
ping google.com
```

---

## `netstat`

Shows open ports and connections.

### Example

```bash id="u3w2zq"
netstat -tulnp
```

---

## `ss`

Modern replacement for netstat.

### Example

```bash id="e8m1ot"
ss -tulnp
```

---

## `nslookup`

Checks DNS resolution.

### Example

```bash id="w6f4rb"
nslookup google.com
```

# 4. What is Patching?

## Simple Definition

Patching means updating software or operating system to:

* Fix bugs
* Improve security
* Improve performance

---

# Default Patching Commands

These are commonly used patch/update commands in Linux.

---

# Ubuntu / Debian Systems

## Check Updates

```bash id="x4s7nm"
sudo apt update
```

## Install Updates

```bash id="3y6wpa"
sudo apt upgrade -y
```

## Full System Upgrade

```bash id="9q2jht"
sudo apt full-upgrade -y
```

## Remove Unused Packages

```bash id="7p1kru"
sudo apt autoremove -y
```

---

# RHEL / CentOS Systems

## Check and Install Updates

```bash id="2d8mfx"
sudo yum update -y
```

OR

```bash id="8r5cql"
sudo dnf update -y
```

---

# SUSE Linux

## Update Packages

```bash id="0n7sde"
sudo zypper update
```

---

# Security Patch Example

## Ubuntu

```bash id="3h0jfw"

```

## RHEL

```bash id="9k6qre"

```
